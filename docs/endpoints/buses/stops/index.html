Bus Stops

Returns a list of all bus stops on the island.

GET /v1/bus/stops

Response​

  • 200 (OK)
  • Schema

Root​

KeyTypeDescription
resultsArrayArray of BusStops. See example response.

BusStop​

KeyTypeDescription
idUUIDA unique ID to represent this bus stop
nameStringThe bus stop name
codeStringThe 4-digit code the uniquely represents this bus stop
latitudeNumberThe latitude of the bus stop
longitudeNumberThe longitude of the bus stop
shelterBooleanTrue if the bus stop has a shelter, otherwise false
