# Parking Spaces Over Time
There are two endpoints below for this chart.

:::info NOTE
To retrieve the current available spaces for all car parks, refer to the [Parking Spaces (Live)](/docs/endpoints/carparks/parking-spaces) documentation.
:::

## Fetching Dates
Provides a list of dates when parking space information was collected. This can be used to populate a date picker.

```
GET /v1/carpark/spaces/dates
```

### Response
This is a sample response.

```json title="Status: 200 (OK)"
{
    "results": [
      "2025-09-25",
      "2025-09-24",
      "2025-09-23",
      "2025-09-22",
      "2025-09-21",
      "2025-09-20",
      "2025-09-19"
    ]
}
```

## Fetch All Info For Date
Returns a list of all space information collected for a specific date, this can populate a chart.

```
GET /v1/carpark/spaces?date={DATE}
```
