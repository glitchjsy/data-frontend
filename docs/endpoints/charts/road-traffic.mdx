# Road Traffic

Returns information about road traffic on the overpass and tunnel between 2019 and 2022.
Tunnel movements were only collected until 08/11/2020

```
GET /v1/charts/road-traffic
```

:::info NOTE
This data is only used for [this chart](/charts/transport/road-traffic) and does not necessarily conform to the same design patterns as the rest of the API.
:::

### Response
<Tabs>
<TabItem value="res-sample-200" label="200 (OK)">

```json title="Status: 200 (OK)"
{
    "results": [
        {
          "date": "2019-01-13",

        {

          "tunnelMovements": "",
```

</TabItem>

<TabItem value="res-schema" label="Schema">

#### Root
| Key | Type | Description |
|-----|------|-------------|
|`results`|Array|An array of `RoadTraffic` records|

#### RoadTraffic
| Key | Type | Description |
|-----|------|-------------|
|`date`|String|The week-ending date of the traffic count (ISO 8601 format, e.g. `"2019-01-13"`)|
|`tunnelMovements`|String|Number of vehicle movements through the tunnel (may be empty if data not collected)|
|`overpassMovements`|String|Number of vehicle movements over the overpass|
